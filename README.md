## What's included
Within the download you'll find the following directories and files, logically grouping common assets and providing both regular and minified variations.

* css
  * daya.css
  * daya.min.css
* js
  * daya.js
  * daya.min.js
  * jQuery-3.4.1.js
  * jQuery-3.4.1.min.js
* fonts
  * icomoon.eot
  * icomoon.svg
  * icomoon.ttf
  * icomoon.woff
